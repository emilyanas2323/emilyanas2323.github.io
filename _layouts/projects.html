Stardew Valley API

Created an API using Rails that lets users retrieve information about the game using a command line user interface created with React & Redux. The information is initially retrieved from the game’s Wiki using a Python web scraper and stored using Postgres. This API is a faster and more organized alternative to browsing the Wiki while playing.

DocLog

DocLog is a tool developed using React, Symptom Checker API, and Cloud Firestore that improves the delivery of medical services to remote communities. Patients log their mood and symptoms and the API suggests other related symptoms. The doctor’s dashboard presents the patient's symptoms with possible diagnoses and can send a message to the patient.
